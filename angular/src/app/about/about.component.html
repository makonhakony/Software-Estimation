About Software Estimation

Current state of modern world

Estimation is building based on the requirements of the technologies shortage field of estimation.

Our core values are building the website for users estimating and collecting data for software evaluation as well as creating storages for user assessment results.

The fact that there are just a few websites that are able to calculate the size and number lines of code in the software project. However, they just have one or few features integrating in the program only. Furthermore, they have such an old interface that users will get bored when they proceed every step to evaluate their software weight and points.

That is the reason why we create this website to integrate a variety of estimation features which can help users have a better scenario of their project quality and quantity. In addition, we provide a website which has a more modern and perspicuous user interface that can improve user experience when using the website as well.


Our passion

Building the application which helps users can estimate the software application easier, such as student, teacher, employee in the company and especially HR of the company.

From that, they can have some information about the application which they want to estimate including the size and source lines of code, use case point, function point.

On top of that, users can recognize the number of members working on that project and the amount of time that they spend on.
